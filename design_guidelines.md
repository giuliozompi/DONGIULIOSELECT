# Руководство по дизайну: Don Giulio Select Telegram Mini App

## Подход к дизайну

**Специализированный подход для Telegram Mini App**
Приложение использует нативную интеграцию с Telegram WebApp API, следуя строгим требованиям платформы с современным минималистичным стилем e-commerce.

### Ключевые принципы
- **Нативность Telegram**: Полная адаптация к UI/UX паттернам Telegram
- **Минимализм**: Чистый дизайн с обильным whitespace ("воздухом")
- **Адаптивность**: Mobile-first подход для всех разрешений
- **Функциональность**: Приоритет удобству и скорости использования

## Цветовая палитра

### Светлая тема (Light Mode)
- **Фон**: `themeParams.bg_color` (системный цвет Telegram)
- **Вторичный фон**: `themeParams.secondary_bg_color`
- **Акцентный цвет**: `themeParams.button_color` (для CTA, выбранных элементов)
- **Текст**: `themeParams.text_color`
- **Акцентный текст**: `themeParams.accent_text_color` (цены, важные элементы)
- **Ссылки**: `themeParams.link_color`

### Темная тема (Dark Mode)
- Автоматическая адаптация всех элементов к темным цветам Telegram
- Использование тех же параметров темы для консистентности

### Семантические цвета
- **Успех**: Зеленый оттенок для подтверждений (✅ Добавлено)
- **Предупреждение**: Желтый для важных уведомлений
- **Ошибка**: Красный для ошибок валидации

## Типографика

### Шрифты
- **Основной**: Системный шрифт устройства/Telegram (`-apple-system, BlinkMacSystemFont, "Segoe UI", Roboto`)
- **Акцентный**: Жирный вариант системного шрифта для заголовков

### Иерархия размеров
- **H1** (Название продукта): `text-2xl font-bold` (24px)
- **H2** (Заголовки секций): `text-xl font-semibold` (20px)
- **H3** (Подзаголовки): `text-lg font-medium` (18px)
- **Цена**: `text-3xl font-bold` (30px) - самый крупный элемент кроме фото
- **Основной текст**: `text-base` (16px)
- **Мелкий текст** (категории, артикулы): `text-sm text-gray-500` (14px)
- **Мельчайший текст**: `text-xs` (12px)

### Стили
- Категории: тонкий шрифт, серый цвет
- Цены: жирный шрифт, `themeParams.accent_text_color`
- Старая цена: зачеркнутая, меньший размер

## Система отступов

Используем Tailwind spacing с акцентом на "воздух":
- **Компактные**: `p-2, p-4` (8px, 16px) для мелких элементов
- **Стандартные**: `p-6, p-8` (24px, 32px) для секций
- **Щедрые**: `py-12, py-16` (48px, 64px) между крупными блоками
- **Margin**: `mb-4, mb-6, mb-8` для вертикального ритма

## Компоненты

### 1. Карточка продукта (Детальная страница)

**Галерея фото**
- Полноширинная карусель с горизонтальным свайпом
- Индикаторы-точки внизу (`flex justify-center gap-2`)
- Высота: `h-80` или `aspect-square`
- Скругление: `rounded-none` (на весь экран)

**Заголовок и метаданные**
- Название: крупное, жирное (`text-2xl font-bold`)
- Категория: мелким шрифтом, серым (`text-sm text-gray-500`)
- Артикул: в блоке деталей или футере

**Блок цены**
- Самый визуально заметный элемент
- Текущая цена: `text-3xl font-bold` + `themeParams.accent_text_color`
- Старая цена (если есть): зачеркнутая, `text-xl line-through opacity-60`
- Единица измерения рядом с ценой (`text-lg`)

**Выбор вкуса/вариаций**
- Кнопки-чипы (`rounded-full px-4 py-2`)
- Активный: `bg-[themeParams.button_color] text-white`
- Неактивный: `border border-gray-300 bg-transparent`
- Недоступный: `opacity-50 cursor-not-allowed`

**Рейтинг вкуса**
- Три кнопки-чипа: "Вкусное", "Очень вкусное", "Вообще супер"
- Выбранный: контрастный цвет (`themeParams.button_color`)
- Статистика под кнопками: `text-sm text-gray-600`

### 2. Аккордеоны

**Структура**
- Заголовок: `flex justify-between items-center py-4 border-b`
- Иконка стрелки: поворот на 180° при открытии
- Контент: `py-4` с анимацией раскрытия

**Секции аккордеонов**
1. Полное описание
2. Пищевая ценность (БЖУ в виде таблицы)
3. Состав
4. Применение/Сочетания

### 3. Блоки рекомендаций

**Горизонтальный скролл**
- Container: `overflow-x-auto flex gap-4 pb-4`
- Мини-карточки: `min-w-[160px] flex-shrink-0`
- Фото: `aspect-square rounded-lg`
- Название: `text-sm font-medium line-clamp-2`
- Цена: `text-base font-bold`

**Типы блоков**
- Связанные продукты
- Связанные рецепты
- Связанные видео (превью с иконкой play)

### 4. Нативные элементы Telegram

**MainButton (CTA)**
- Всегда внизу экрана
- Текст: "Добавить [Вес/Кол-во] в Корзину за [Цена]₽"
- После добавления: "✅ Добавлено" с зеленым цветом
- Haptic Feedback при клике

**BackButton**
- Нативная кнопка Telegram для навигации назад

### 5. Каталог и навигация

**Трехуровневая структура**
- Level 1: Табы или список категорий (`flex gap-2 overflow-x-auto`)
- Level 2-3: Вложенные списки с хлебными крошками
- Превью карточки: фото + название + цена

**Карточки превью**
- Grid: `grid grid-cols-2 gap-4` на мобильном
- Фото: `aspect-square rounded-lg`
- Название: `text-sm font-medium line-clamp-2 mt-2`
- Цена: `text-base font-bold mt-1`

### 6. Корзина

**Список товаров**
- Каждый товар: горизонтальная карточка
- Миниатюра фото слева (`w-20 h-20 rounded`)
- Название и параметры справа
- Контроллер количества: `flex items-center gap-3`
- Кнопки +/- : `w-8 h-8 rounded-full border`

**Итоговый блок**
- Фиксирован внизу
- Общая сумма: `text-2xl font-bold`
- Кнопка оформления: нативный MainButton

### 7. Колесо Фортуны (Геймификация)

**Визуальное представление**
- Canvas/SVG круг разделенный на секторы
- Яркие цвета для каждого сектора
- Центральная кнопка "Крутить"
- Индикатор вверху для определения выигрыша

**Призы**
- Иконка/изображение приза
- Название
- Тип (скидка, подарок, купон)

### 8. AI Ассистент

**Интерфейс чата**
- Модальное окно или отдельная страница
- Пузыри сообщений: округлые (`rounded-2xl`)
- Сообщения пользователя: справа, `bg-[themeParams.button_color]`
- Ответы ассистента: слева, `bg-gray-100 dark:bg-gray-800`
- Поле ввода внизу с кнопкой отправки

## Анимации

**Минимальное использование**
- Haptic Feedback для ключевых действий (не визуальная анимация)
- Плавные переходы для аккордеонов (`transition-all duration-300`)
- Smooth scroll для горизонтальных списков
- Индикатор загрузки для асинхронных операций

## Изображения

### Продуктовые фото
- Высококачественные фото продуктов на белом/нейтральном фоне
- Формат: квадратный (`aspect-square`)
- Множественные ракурсы в галерее

### Рецепты и видео
- Превью рецептов: готовое блюдо
- Превью видео: кадр с иконкой play по центру

### Иконки
- Использовать Font Awesome или Heroicons via CDN
- Размер: `w-5 h-5` или `w-6 h-6` для стандартных иконок

## Accessibility & UX

- Адаптация к темной/светлой теме Telegram автоматическая
- Все интерактивные элементы минимум `h-11` (44px) для удобного тапа
- Haptic feedback для важных действий
- Индикаторы загрузки для асинхронных операций
- Четкие состояния ошибок с понятными сообщениями
